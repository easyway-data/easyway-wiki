# Vision

The strategic foundation of the EasyWay ecosystem.

We started building a data portal for small businesses. Along the way, we accidentally built a methodology for agentic software development, three deterministic engines, a governance framework, and an ethical decision system for AI agents. This section tells that story and maps every product that emerged.

## Key Concepts

- **Three Circles**: Private → Source-Available → Open Source. Maturity is earned, not declared.
- **Brain vs Muscles**: AI (EasyWay) handles intelligence. Deterministic engines (HALE-BOPP) handle precision.
- **GEDI**: The ethical decision framework. Never blocks, always illuminates. The cricket on your shoulder.
- **Multi-Agent Future**: Many AIs will work on the same codebase. Each repo = one boundary, one contract, one product.
